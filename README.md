# 👨‍💻 About Me

AI에게 프로그래밍의 제어권을 넘겨주지 않기 위해 치열하게 싸우는 중입니다.
<br/>
클래스와 알고리즘 그리고 아키텍처 관점에서 본질에 대한 고찰로 깊이있게 코드를 이해하기 위해 노력합니다.
<br/><br/>
새로운 기술을 탐구하고, 동시에 지식을 다듬어 가며 
<br/>
성장에 재미와 희열은 지속적인 추진력을 낼 수 있는 연료가 됩니다.
<br/><br/>
처음엔 낯설었던 “개발자”라는 타이틀이 이제는 뚜렷한 정체성이 되었고,
<br/>
꾸준한 학습과 실무 경험을 통해 기술적으로 깊이 있는 개발을 할 수 있는 단계로 점차 나아가고 있습니다.
<br/>
